## Get started

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/TheAlgorithms/Go.git
    cd Go
    ```
  </Step>
  <Step title="Run the tests">
    Verify all algorithms work correctly on your machine.
    ```bash
    go test ./...
    ```
  </Step>
  <Step title="Explore an algorithm">
    Import and use any algorithm package directly in your Go code.
    ```go
    import "github.com/TheAlgorithms/Go/sort"

    sorted := sort.Bubble([]int{5, 2, 8, 1, 9})
    ```
  </Step>
  <Step title="Contribute">
    Add a new algorithm or improve an existing one. See our [contribution guide](/contributing) for guidelines.
  </Step>
</Steps>

<Note>
  All implementations are for **educational purposes**. They follow Go best practices, include GoDoc comments, and are covered by tests. The module path is `github.com/TheAlgorithms/Go`.
</Note>
