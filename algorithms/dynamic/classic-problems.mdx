  <Accordion title="Knapsack — 0/1 knapsack problem">

    Given a knapsack with a weight capacity and a list of items (each with a weight and a value), determine the maximum total value that can be packed without exceeding the weight limit. Each item can be taken at most once (0/1 constraint).

    ```go
    func Knapsack(maxWeight int, weights, values []int) int
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `maxWeight` | `int` | Maximum weight the knapsack can hold |
    | `weights` | `[]int` | Weight of each item |
    | `values` | `[]int` | Value of each item |

    **Returns:** the maximum achievable value.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        maxWeight := 50
        weights := []int{10, 20, 30}
        values  := []int{60, 100, 120}

        profit := dynamic.Knapsack(maxWeight, weights, values)
        fmt.Println(profit) // 220
    }
    ```

    <Tip>
    Items with indices 0 and 1 (weights 10 and 20, values 60 and 100) are selected, giving a total weight of 30 and total value of 160. Adding item 2 (weight 30) would exceed 50, so the optimal is items 0 and 2: weight 40, value 180. Actually, all three items weigh 60 which exceeds the limit—the optimal pick is items 1 and 2: weight 50, value 220.
    </Tip>

    **Complexity:** Time O(n × W), Space O(n × W), where n is the number of items and W is `maxWeight`.

  </Accordion>

  <Accordion title="CoinChange — count combinations">

    Count the number of ways to make change for a given `amount` using an unlimited supply of coins with the given denominations.

    ```go
    func CoinChange(coins []int32, amount int32) int32
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `coins` | `[]int32` | Available coin denominations |
    | `amount` | `int32` | Target amount to make change for |

    **Returns:** the number of distinct combinations that sum to `amount`.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        coins  := []int32{1, 2, 3}
        amount := int32(5)

        ways := dynamic.CoinChange(coins, amount)
        fmt.Println(ways) // 5
    }
    ```

    **Complexity:** Time O(n × m), Space O(m), where n is the number of coin denominations and m is `amount`.

  </Accordion>

  <Accordion title="NthFibonacci — iterative Fibonacci">

    Returns the n-th Fibonacci number using an O(1)-space iterative approach (bottom-up DP with two rolling variables).

    ```go
    func NthFibonacci(n uint) uint
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `n` | `uint` | Index in the Fibonacci sequence (0-indexed) |

    **Returns:** the n-th Fibonacci number, where F(0) = 0 and F(1) = 1.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        fmt.Println(dynamic.NthFibonacci(0))  // 0
        fmt.Println(dynamic.NthFibonacci(1))  // 1
        fmt.Println(dynamic.NthFibonacci(10)) // 55
    }
    ```

    **Complexity:** Time O(n), Space O(1).

  <Accordion title="TrapRainWater — elevation map water trapping">

    Given an elevation map represented as an array of non-negative integers, compute the total amount of rainwater that can be trapped between the bars.

    ```go
    func TrapRainWater(height []int) int
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `height` | `[]int` | Elevation map where each element is the bar height at that position |

    **Returns:** the total units of trapped rainwater.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        height := []int{0, 1, 0, 2, 1, 0, 1, 3, 2, 1, 2, 1}
        fmt.Println(dynamic.TrapRainWater(height)) // 6
    }
    ```

    The algorithm precomputes `leftMax[i]` and `rightMax[i]` arrays, then accumulates `min(leftMax[i], rightMax[i]) - height[i]` at each position.

    **Complexity:** Time O(n), Space O(n).

  </Accordion>

  <Accordion title="IsSubsetSum — subset sum decision problem">

    Given a set of non-negative integers, determine whether any subset of the given set sums to the target value.

    ```go
    func IsSubsetSum(array []int, sum int) (bool, error)
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `array` | `[]int` | Set of non-negative integers |
    | `sum` | `int` | Target sum (must be non-negative) |

    **Returns:** `(true, nil)` if a subset exists, `(false, nil)` if no subset sums to `sum`, or `(false, error)` if `sum` is negative.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        set := []int{3, 34, 4, 12, 5, 2}

        ok, err := dynamic.IsSubsetSum(set, 9)
        fmt.Println(ok, err) // true <nil>   (subset {4, 5})

        ok, err = dynamic.IsSubsetSum(set, 30)
        fmt.Println(ok, err) // false <nil>
    }
    ```

    **Complexity:** Time O(n × sum), Space O(n × sum).

  </Accordion>

  <Accordion title="CutRodRec / CutRodDp — rod cutting">

    Given a rod of a specified length and a price table indexed by cut length, find the maximum revenue obtainable by cutting the rod into pieces. Two implementations are provided: a naive recursive approach and an optimized DP approach.

    ```go
    func CutRodRec(price []int, length int) int
    func CutRodDp(price []int, length int) int
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `price` | `[]int` | Price table where `price[i]` is the price of a rod of length `i` (index 0 unused) |
    | `length` | `int` | Total length of the rod to cut |

    **Returns:** the maximum revenue.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        // price[i] = price of rod of length i; price[0] is unused
        price  := []int{0, 1, 5, 8, 9, 17, 17, 17, 20, 24, 30}
        length := 10

        fmt.Println(dynamic.CutRodRec(price, length)) // 30
        fmt.Println(dynamic.CutRodDp(price, length))  // 30
    }
    ```

    <Note>
    `CutRodRec` has exponential time complexity due to repeated subproblem recomputation. Prefer `CutRodDp` for any practical input.
    </Note>

    **Complexity:**
    - `CutRodRec`: Time O(2ⁿ), Space O(n) recursion stack
    - `CutRodDp`: Time O(n²), Space O(n)

  </Accordion>

  <Accordion title="MatrixChainRec / MatrixChainDp — matrix chain multiplication">

    Given a sequence of matrices, find the most efficient way to multiply them together (minimizing the total number of scalar multiplications). The matrices themselves are not multiplied; only the optimal parenthesization cost is computed.

    ```go
    func MatrixChainRec(D []int, i, j int) int
    func MatrixChainDp(D []int) int
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `D` | `[]int` | Dimension array where matrix `i` has dimensions `D[i-1] × D[i]` |
    | `i`, `j` | `int` | Start and end indices for `MatrixChainRec` (use `1` and `len(D)-1`) |

    **Returns:** the minimum number of scalar multiplications needed.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        // 4 matrices with dimensions: 2×2, 2×2, 2×2, 2×2
        D := []int{2, 2, 2, 2, 2}

        rec := dynamic.MatrixChainRec(D, 1, len(D)-1)
        dp  := dynamic.MatrixChainDp(D)
        fmt.Println(rec, dp) // 24 24
    }
    ```

    **Complexity:**
    - `MatrixChainRec`: exponential time without memoization
    - `MatrixChainDp`: Time O(n³), Space O(n²)

  </Accordion>

  <Accordion title="NthCatalanNumber — Catalan numbers">

    Returns the n-th Catalan number. Catalan numbers count many combinatorial structures, including the number of valid parenthesizations of n+1 factors, the number of rooted binary trees with n+1 leaves, and more.

    ```go
    func NthCatalanNumber(n int) (int64, error)
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `n` | `int` | Index of the Catalan number to compute (must be ≥ 0) |

    **Returns:** `(C_n, nil)` on success, or `(0, error)` if `n` is negative.

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        for i := 0; i <= 6; i++ {
            c, _ := dynamic.NthCatalanNumber(i)
            fmt.Printf("C(%d) = %d\n", i, c)
        }
        // C(0) = 1
        // C(1) = 1
        // C(2) = 2
        // C(3) = 5
        // C(4) = 14
        // C(5) = 42
        // C(6) = 132
    }
    ```

    **Complexity:** Time O(n²), Space O(n).

  </Accordion>

  <Accordion title="Bin2 — binomial coefficient">

    Computes the binomial coefficient C(n, k) — the number of ways to choose k elements from a set of n elements — using the Pascal's triangle recurrence: `C(n, k) = C(n-1, k-1) + C(n-1, k)`.

    ```go
    func Bin2(n int, k int) int
    ```

    | Parameter | Type | Description |
    |---|---|---|
    | `n` | `int` | Total number of elements |
    | `k` | `int` | Number of elements to choose |

    **Returns:** C(n, k).

    ```go
    package main

    import (
        "fmt"
        "github.com/TheAlgorithms/Go/dynamic"
    )

    func main() {
        fmt.Println(dynamic.Bin2(5, 2))  // 10
        fmt.Println(dynamic.Bin2(10, 3)) // 120
        fmt.Println(dynamic.Bin2(50, 5)) // 2118760
    }
    ```

    **Complexity:** Time O(n × k), Space O(n × k).

  </Accordion>
